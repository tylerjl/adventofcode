Advent of Code

Haskell Solutions

Last year, @topaz ran a coding challenge site called Advent of Code over the holidays. Though I initially completed a few in my lingua franca (ruby) just for fun, some challenges presented some interesting problems that forced solutions that were a bit more engineered than a few lines of script. I decided to try and solve all of them in Haskell in order to stretch myself as I haven't done a whole lot of work in (functional|pure|lazy) languages very much, if at all.

I took advantage of a small project with only myself at the helm to fully explore different aspects of the Haskell ecosystem: stack (the new Haskell build tool), various testing and benchmarking libraries, and some high-performance libraries like Repa to understand what high-performing Haskell code looks like. Some of what you'll see is very much overkill, but it touches most aspects of the Haskell ecosystem in an education and demonstrative way.

Source

All of the source code is hosted on GitHub. In addition, the README is pretty straightforward if you want to build it yourself: stack makes the process very convenient.

Documentation

The Haskell documentation tool Haddock is the de facto documentation generation tool, so the solution HTML documentation is provided as an example of what that process looks like. Note the Source links accompanying the functions and datatypes if you're curious about what inline Haddock documentation looks like.
